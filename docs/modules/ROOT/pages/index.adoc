== Installation

If you want to use this extension, you need to add the `io.quarkiverse.dapr:quarkus-dapr` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.dapr</groupId>
    <artifactId>quarkus-dapr</artifactId>
</dependency>
----
